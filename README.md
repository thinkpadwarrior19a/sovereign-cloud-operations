# Sovereign Cloud Operations

This repository contains the manuscript and supporting materials for the book
**“Sovereign Cloud Operations: Agentic AI, Observability, and Automation for the Fully Sovereign, Multi‑Cloud Enterprise.”**

It builds on the architectural foundations of *Zero‑Copy Integration* and focuses on:

- Sovereign‑aware observability and network performance
- IT automation and infrastructure as code
- Agentic AI operations using IBM Concert and IBM watsonx Orchestrate
- AI governance with IBM watsonx.governance
- Human factors, shift‑left operations, and multi‑cloud resilience

## Structure

- `book/` – chapter markdown files
- `images/` – diagrams and figures
- `config/` – tooling config (Jekyll, LaTeX preamble, etc.)
- `scripts/` – helpers to build PDF/EPUB
